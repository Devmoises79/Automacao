# 🕹 Automação Desktop — Python (PyAutoGUI)

📖 Sobre o Projeto

Este projeto consiste em uma automação simples de tarefas no computador utilizando Python, com foco em simular interações humanas no sistema operacional.

A aplicação utiliza a biblioteca PyAutoGUI para controlar mouse e teclado, automatizando ações como abrir programas, navegar na web e realizar pesquisas automaticamente.

# 🎯 Objetivos

- Praticar automação de interfaces gráficas (GUI)
- Simular interação humana com o computador
- Utilizar controle de tempo para execução de ações
- Explorar automação de navegação em navegador
- Trabalhar com posicionamento de tela (coordinates)

# ⚙️ Tecnologias Utilizadas

- Python 3
- PyAutoGUI
- Time (biblioteca padrão)
- Pyperclip (em algumas interações)

# 🧩 Funcionalidades

- ⌨ Automação de teclado (digitação automática)
- 🖱 Controle de mouse por coordenadas
- 🌐 Abertura automática de navegador
- 🔎 Execução de pesquisas no YouTube
- ⏱ Controle de tempo entre ações (sleep/pause)
- 📌 Como Funciona

O script simula o comportamento de um usuário real:

- Abre o menu iniciar
- Pesquisa e abre o Chrome
- Acessa o YouTube
- Realiza uma busca automática
- Clica e interage com a interface via coordenadas

# 🧠 Aprendizados

Este projeto reforça conceitos importantes como:

- Automação de tarefas repetitivas
- Interação com GUI via código
- Controle de tempo de execução
- Coordenação de ações sequenciais
- Simulação de comportamento humano em sistemas

# 🚀 Execução

```bash
pip install pyautogui pyperclip
python automacao.py
```

⚠️ Importante: o script depende da resolução da tela e posição dos elementos na interface.
